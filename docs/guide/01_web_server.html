🌐 Apache 웹 서버 설치 가이드

✅ 1단계. Apache 설치

sudo apt update
sudo apt install apache2 -y

설치 후 서비스 상태 확인:

sudo systemctl status apache2

✅ active (running) 이라고 나오면 정상입니다.

✅ 2단계. Apache2 정상 배포 확인

curl -I localhost:80

HTTP 200 OK 가 뜨면 정상

✅ 3단계. Apache2 시스템 데몬 서비스 등록 (서버 재시작 시 실행 되게)

sudo systemctl start apache2
sudo systemctl enable apache2
